# 인시던트 대응 런북

## 장애 레벨 정의
- Sev1: 주문/업로드/어드민 핵심 기능이 중단
- Sev2: 일부 기능 저하, 대체 경로 존재
- Sev3: 경미한 오류/로그 증가

## 즉시 조치 순서
1. 증상 확인 (로그/모니터링)
2. 영향 범위 판단 (주문/업로드/admin)
3. 임시 완화 (기능 토글이 있으면 사용)
4. 롤백/핫픽스 기준 판단

## 롤백 기준
- 5xx 오류 급증
- 주문/업로드 불가
- admin 접근 불가
- 롤백이 더 빠르면 롤백 우선

## 롤백 방법 (Vercel 기준)
- 최근 정상 배포로 revert
- 롤백 후 `verify-deploy` 실행으로 기본 경로 확인

## DB 롤백 원칙
- 원칙적으로 DB 롤백은 수행하지 않는다
- 필요 시 마이그레이션/핫픽스로 복구한다
