# 런칭 후 48시간 운영 체크리스트

## 핵심 지표 (정량/정성)
- 주문 생성 성공률(체크아웃)
- 커스텀 요청 성공률
- 업로드 실패율(presign 실패 vs 업로드 PUT 실패)
- admin 401/403 비정상 증가
- 429(rate limit) 발생량
- 알림 누락 빈도

## 로그에서 찾아야 할 패턴
- request id 기준으로 요청 흐름을 추적
- endpoint별 주요 에러 코드와 원인 후보 정리
- 에러 급증 구간의 배포/환경 변수 변경 여부 확인

## 운영 루틴 (1일차)
- 1시간마다: 주문/커스텀 성공률, 429 발생량, 알림 누락 확인
- 6시간마다: 업로드 실패율, admin 401/403 증가 확인
- 1회 정리: 주요 장애/문의 요약 및 대응 결과 기록

## 운영 루틴 (2일차)
- 1시간마다: 전일 대비 성공률/실패율 추세 확인
- 6시간마다: 로그 샘플링으로 반복 오류 패턴 확인
- 1회 정리: 장애 유형별 재발 방지 항목 정리
